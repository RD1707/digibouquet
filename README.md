# para você, flores.

*um jardim digital interativo.*

Um projeto minimalista criado para construir e eternizar buquês de flores digitais. Feito com muito carinho como um presente especial para celebrar o amor e o tempo juntos.

## sobre o projeto

A aplicação permite a criação de buquês personalizados em um ambiente interativo e fluído. Cada buquê criado pode ser guardado em um "jardim" digital (galeria), criando um arquivo de memórias.

**Principais funcionalidades:**
- Montagem interativa de buquês (modos colorido e monocromático).
- Jardim digital para visualização do histórico de flores criadas.
- Contador de dias personalizado para datas especiais.
- Interface responsiva, limpa e minimalista.

## stack tecnológica

- **Frontend:** React, Vite, TypeScript
- **Estilização:** Tailwind CSS, componentes Radix UI (shadcn/ui)
- **Roteamento:** React Router Dom
- **Backend/Database:** Supabase
- **Gerenciamento de Estado/Fetch:** React Query
